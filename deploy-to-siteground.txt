SITEGROUND DEPLOYMENT INSTRUCTIONS
==================================

Since we're having SSH key issues, here are two ways to deploy:

METHOD 1: Through SiteGround File Manager (Easiest)
---------------------------------------------------
1. Log into SiteGround Site Tools
2. Go to Site > File Manager
3. Navigate to /public_html/
4. Click "Upload" button
5. Upload these files and folders:
   - index.html
   - styles.css
   - script.js
   - admin.html
   - admin-styles.css
   - admin.js
   - newsletter-form.html
   - images/ (entire folder with all book covers)

METHOD 2: Re-initialize Git on SiteGround
-----------------------------------------
1. In SiteGround Site Tools, go to Devs > Git
2. Delete the existing repository if it shows one
3. Create a new Git repository pointing to /public_html/
4. It will give you a new Git URL
5. Then run these commands locally:

   cd "C:\CODING\LARGER THEN LIFE MOCKUP"
   git remote remove siteground
   git remote add siteground [NEW-GIT-URL-FROM-SITEGROUND]
   git push siteground master

METHOD 3: Using FTP Client
--------------------------
1. Use an FTP client like FileZilla
2. Connect with:
   - Host: ftp.largerthanlifecomics.com
   - Username: [your FTP username]
   - Password: [your FTP password]
   - Port: 21
3. Upload all files to /public_html/

IMPORTANT FILES TO UPLOAD:
- index.html (main page)
- styles.css (styling)
- script.js (interactions)
- admin.html (admin panel)
- admin-styles.css (admin styling)
- admin.js (admin functionality)
- newsletter-form.html (standalone newsletter page)
- images/ folder containing:
  - cork-1-cover.jpeg
  - cork-2-3-cover.jpeg
  - cork-4-5-cover.jpeg
  - prodigy-cover.jpeg
  - new-dawn-cover.jpeg
  - loneliness-fix-cover.jpg

After uploading, your site should be live at largerthanlifecomics.com!
